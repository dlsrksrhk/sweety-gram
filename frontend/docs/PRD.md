# Sweety Gram MVP PRD

## 🎯 핵심 정보

**목적**: 사진과 짧은 글을 피드 형식으로 공유하고, 팔로우 기반으로 관심 있는 사람들의 콘텐츠를 한눈에 소비할 수 있는 SNS 플랫폼 제공
**사용자**: 일상, 취미, 관심사를 사진 중심으로 공유하고 싶은 개인 사용자

---

## 🚶 사용자 여정

```
1. 랜딩 페이지 (비로그인 상태)
   ↓ [회원가입 버튼 클릭] 또는 [로그인 버튼 클릭]

2. 회원가입 페이지 / 로그인 페이지
   ↓ [인증 성공]

   [신규 사용자] → 홈 피드 페이지 (빈 피드 안내 표시) → 다른 사용자 탐색
   [기존 사용자] → 홈 피드 페이지 (팔로우 중인 계정의 게시물 표시)
   ↓

3. 홈 피드 페이지
   ↓ [게시물 작성] / [게시물 클릭] / [유저 프로필 클릭] / [탐색 탭 클릭]

   [게시물 작성] → 게시물 작성 페이지 → 작성 완료 → 홈 피드 페이지
   [게시물 클릭] → 게시물 상세 페이지 → 댓글/좋아요 → 홈 피드 복귀
   [유저 프로필 클릭] → 유저 프로필 페이지 → 팔로우/언팔로우
   [탐색 탭 클릭] → 탐색 페이지 → 게시물 클릭 → 게시물 상세 페이지
   ↓

4. 내 프로필 페이지
   ↓ [게시물 클릭] 또는 [팔로워/팔로잉 클릭]

   [내 게시물 클릭] → 게시물 상세 페이지 → 수정/삭제 가능
   [팔로워/팔로잉 클릭] → 팔로우 목록 페이지 → 유저 프로필 페이지

5. 완료 → [홈 피드로 돌아가기] / [추가 게시물 작성] / [탐색 계속하기]
```

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|--------------|------------|
| **F001** | 게시물 작성 | 이미지 1장 + 텍스트 캡션으로 게시물 생성 | SNS의 핵심 콘텐츠 생산 기능 | 게시물 작성 페이지 |
| **F002** | 홈 피드 조회 | 팔로우한 계정의 최신 게시물을 시간순으로 표시 | 콘텐츠 소비의 핵심 진입점 | 홈 피드 페이지 |
| **F003** | 게시물 상세 조회 | 단일 게시물의 이미지, 캡션, 댓글, 좋아요 표시 | 게시물 상호작용의 중심 화면 | 게시물 상세 페이지 |
| **F004** | 게시물 수정/삭제 | 본인 게시물의 캡션 수정 또는 게시물 전체 삭제 | 콘텐츠 관리 최소 기능 | 게시물 상세 페이지 |
| **F005** | 좋아요 | 게시물에 좋아요 토글 및 좋아요 수 표시 | 사용자 간 가장 기본적인 반응 수단 | 홈 피드 페이지, 게시물 상세 페이지 |
| **F006** | 댓글 작성/삭제 | 게시물에 댓글 작성 및 본인 댓글 삭제 | 사용자 간 텍스트 상호작용 수단 | 게시물 상세 페이지 |
| **F007** | 팔로우/언팔로우 | 다른 사용자를 팔로우하거나 언팔로우 | 피드 큐레이션의 핵심 메커니즘 | 유저 프로필 페이지, 팔로우 목록 페이지 |
| **F008** | 유저 프로필 조회 | 특정 사용자의 프로필 사진, 소개, 게시물 목록 조회 | 다른 사용자 탐색 및 팔로우 결정 근거 | 유저 프로필 페이지 |
| **F009** | 내 프로필 조회 | 본인의 프로필 사진, 소개, 게시물 목록 조회 | 내 콘텐츠 관리 및 팔로우 현황 확인 | 내 프로필 페이지 |
| **F010** | 내 프로필 수정 | 프로필 사진, 이름, 한 줄 소개 수정 | 사용자 정체성 표현의 최소 기능 | 내 프로필 페이지 |
| **F011** | 팔로워/팔로잉 목록 조회 | 나 또는 다른 사용자의 팔로워/팔로잉 목록 확인 | 소셜 관계망 파악 및 새 사용자 탐색 | 팔로우 목록 페이지 |
| **F012** | 탐색 피드 | 전체 공개 게시물을 최신순으로 탐색 | 팔로우하지 않은 새 콘텐츠 발견 수단 | 탐색 페이지 |

### 2. MVP 필수 지원 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|--------------|------------|
| **F013** | 회원가입 | 이메일 + 비밀번호 + 사용자명으로 계정 생성 | 서비스 이용을 위한 최소 인증 | 회원가입 페이지 |
| **F014** | 로그인 | 이메일 + 비밀번호로 세션 생성 | 서비스 이용을 위한 최소 인증 | 로그인 페이지 |
| **F015** | 로그아웃 | 현재 세션 종료 | 계정 보안 최소 기능 | 홈 피드 페이지 (헤더) |

### 3. MVP 이후 기능 (제외)

- 스토리(24시간 게시물), 릴스(숏폼 영상), 다중 이미지 캐러셀
- 해시태그 검색 및 태그 피드
- 사용자 검색 (검색창)
- 실시간 알림 (좋아요/댓글/팔로우 알림)
- DM (다이렉트 메시지)
- 게시물 저장/북마크, 댓글 좋아요, 대댓글
- 소셜 로그인 (Google, Apple 등)
- 계정 비공개 설정

## 📄 페이지별 상세 기능

### 랜딩 페이지

> **구현 기능:** `F013`, `F014` | **메뉴 위치:** 비로그인 접근 시 자동 표시

| 항목 | 내용 |
|------|------|
| **역할** | 비로그인 사용자에게 서비스를 소개하고 회원가입/로그인으로 유도하는 진입점 |
| **진입 경로** | 비로그인 상태로 서비스 접속 시 자동 리디렉션 |
| **사용자 행동** | 서비스 소개를 확인하고 로그인 또는 회원가입 버튼 클릭 |
| **주요 기능** | • 서비스 슬로건 및 대표 이미지 표시<br>• **로그인** 버튼<br>• **회원가입** 버튼 |
| **다음 이동** | 로그인 클릭 → 로그인 페이지, 회원가입 클릭 → 회원가입 페이지 |

---

### 로그인 페이지

> **구현 기능:** `F014` | **메뉴 위치:** 비로그인 사용자 메뉴 > 로그인

| 항목 | 내용 |
|------|------|
| **역할** | 기존 사용자가 이메일과 비밀번호로 세션을 생성하는 인증 페이지 |
| **진입 경로** | 랜딩 페이지 로그인 버튼 클릭 / 회원가입 페이지 내 로그인 링크 클릭 |
| **사용자 행동** | 이메일과 비밀번호 입력 후 로그인 버튼 클릭 |
| **주요 기능** | • 이메일 입력 필드 (형식 유효성 검사)<br>• 비밀번호 입력 필드 (표시/숨김 토글)<br>• 인증 실패 시 에러 메시지 표시<br>• 회원가입 페이지 이동 링크<br>• **로그인** 버튼 |
| **다음 이동** | 성공 → 홈 피드 페이지, 실패 → 에러 메시지 표시 |

---

### 회원가입 페이지

> **구현 기능:** `F013` | **메뉴 위치:** 비로그인 사용자 메뉴 > 회원가입

| 항목 | 내용 |
|------|------|
| **역할** | 신규 사용자가 이메일, 비밀번호, 사용자명을 입력하여 계정을 생성하는 페이지 |
| **진입 경로** | 랜딩 페이지 회원가입 버튼 클릭 / 로그인 페이지 내 회원가입 링크 클릭 |
| **사용자 행동** | 사용자명, 이메일, 비밀번호 입력 후 회원가입 버튼 클릭 |
| **주요 기능** | • 사용자명 입력 필드 (중복 확인, 영문/숫자/언더스코어)<br>• 이메일 입력 필드 (형식 유효성 검사)<br>• 비밀번호 입력 필드 (최소 8자)<br>• 필드별 실시간 유효성 에러 표시<br>• **회원가입** 버튼 |
| **다음 이동** | 성공 → 홈 피드 페이지, 실패 → 에러 메시지 표시 |

---

### 홈 피드 페이지

> **구현 기능:** `F002`, `F005`, `F015` | **메뉴 위치:** 하단 탭 바 > 홈

| 항목 | 내용 |
|------|------|
| **역할** | 팔로우한 계정들의 최신 게시물을 타임라인 형식으로 보여주는 핵심 화면 |
| **진입 경로** | 로그인 성공 후 자동 이동 / 하단 탭 바 홈 클릭 / 로고 클릭 |
| **사용자 행동** | 피드를 스크롤하며 게시물 소비, 좋아요 누르기, 게시물/프로필 클릭 |
| **주요 기능** | • 팔로우 계정 게시물 최신순 무한 스크롤<br>• 팔로우 없을 시 탐색 페이지 유도 안내<br>• 게시물 카드 (아바타, 이미지, 캡션 미리보기, 좋아요/댓글 수)<br>• **좋아요** 토글 버튼<br>• 헤더 게시물 작성 아이콘 / **로그아웃** 버튼 |
| **다음 이동** | 게시물 클릭 → 게시물 상세 페이지, 아바타 클릭 → 유저 프로필 페이지, 로그아웃 → 랜딩 페이지 |

---

### 게시물 작성 페이지

> **구현 기능:** `F001` | **메뉴 위치:** 헤더 > 게시물 작성 아이콘

| 항목 | 내용 |
|------|------|
| **역할** | 이미지와 캡션을 입력하여 새 게시물을 생성하는 콘텐츠 생산 페이지 |
| **진입 경로** | 헤더의 게시물 작성 아이콘 클릭 (로그인 사용자만 접근) |
| **사용자 행동** | 이미지 업로드 후 캡션 작성, 공유 버튼 클릭으로 게시 |
| **주요 기능** | • 이미지 파일 업로드 (드래그&드롭 또는 파일 선택, JPG/PNG/WEBP)<br>• 이미지 미리보기 표시<br>• 캡션 텍스트 입력 필드 (최대 2,200자)<br>• 이미지 미선택 시 공유 버튼 비활성화<br>• **공유** 버튼 / **취소** 버튼 |
| **다음 이동** | 성공/취소 → 홈 피드 페이지, 실패 → 에러 메시지 표시 |

---

### 게시물 상세 페이지

> **구현 기능:** `F003`, `F004`, `F005`, `F006` | **메뉴 위치:** 컨텍스트 메뉴 > 게시물 클릭

| 항목 | 내용 |
|------|------|
| **역할** | 단일 게시물의 전체 내용(이미지, 캡션, 댓글)을 표시하고 상호작용하는 페이지 |
| **진입 경로** | 홈 피드 / 탐색 / 내 프로필 / 유저 프로필 페이지의 게시물 클릭 |
| **사용자 행동** | 이미지와 캡션 확인, 좋아요/댓글 작성, 본인 게시물 수정/삭제 |
| **주요 기능** | • 게시물 이미지 전체 표시<br>• **좋아요** 토글 버튼 및 좋아요 수<br>• 댓글 목록 및 댓글 입력 필드 + **작성** 버튼<br>• 본인 댓글 **삭제** 버튼<br>• 본인 게시물인 경우 **수정** / **삭제** 버튼 |
| **다음 이동** | 작성자 클릭 → 유저 프로필 페이지, 삭제 완료 → 홈 피드 페이지 |

---

### 탐색 페이지

> **구현 기능:** `F012` | **메뉴 위치:** 하단 탭 바 > 탐색

| 항목 | 내용 |
|------|------|
| **역할** | 팔로우 여부와 관계없이 전체 공개 게시물을 탐색하여 새로운 콘텐츠를 발견하는 페이지 |
| **진입 경로** | 하단 탭 바 탐색 클릭 / 홈 피드 빈 상태 안내 버튼 클릭 |
| **사용자 행동** | 전체 공개 게시물을 그리드로 스크롤하며 관심 게시물 클릭 |
| **주요 기능** | • 전체 공개 게시물 최신순 그리드(3열) 무한 스크롤<br>• 썸네일 위 좋아요 수 / 댓글 수 오버레이<br>• 썸네일 클릭 시 게시물 상세 페이지 이동 |
| **다음 이동** | 게시물 클릭 → 게시물 상세 페이지 |

---

### 내 프로필 페이지

> **구현 기능:** `F009`, `F010`, `F011` | **메뉴 위치:** 하단 탭 바 > 내 프로필

| 항목 | 내용 |
|------|------|
| **역할** | 로그인한 사용자 본인의 프로필과 게시물을 확인하고 프로필을 수정하는 페이지 |
| **진입 경로** | 하단 탭 바 내 프로필 아이콘 클릭 |
| **사용자 행동** | 본인 게시물과 팔로우 현황 확인, 프로필 수정 버튼으로 정보 변경 |
| **주요 기능** | • 프로필 사진, 사용자명, 한 줄 소개 표시<br>• 게시물 수 / 팔로워 수 / 팔로잉 수 (클릭 시 팔로우 목록 이동)<br>• 본인 게시물 그리드(3열) 목록<br>• **프로필 수정** 버튼 (사진 변경, 이름/소개 수정 인라인 폼)<br>• 수정 완료 **저장** 버튼 |
| **다음 이동** | 게시물 클릭 → 게시물 상세 페이지, 팔로워/팔로잉 숫자 클릭 → 팔로우 목록 페이지 |

| 항목 | 내용 |
|------|------|

| **사용자 행동** | 팔로워/팔로잉 탭을 전환하며 목록 확인, 각 사용자 팔로우/언팔로우 |

## 🗄️ 데이터 모델
